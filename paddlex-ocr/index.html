六、问题

1.在gpu上运行paddlex报错：“[Hint: ‘cudaErrorInitializationError’. The API call failed because the CUDA driver and runtime could not be initialized. ]”

解释：使用多进程时，运行异常

解决：将所有与paddlex相关的模块的import放在函数内
